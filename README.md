# 📊 Superstore Sales Analysis using SQL

## 🎯 Project Overview
Complete data analysis of Superstore sales data using MySQL. Analyzed 9,694 transactions, $2.27M in sales.

## 🛠️ Tools Used
- MySQL Workbench
- Superstore Dataset

## 📈 Key Findings

| Metric | Value |
|:---|:---|
| Total Sales | $2,272,450 |
| Total Profit | $282,858 |
| Profit Margin | 12.45% |
| Best Category | Technology (17.39%) |
| Worst Category | Furniture (2.32%) |

### 💰 Discount Impact
| Discount | Avg Profit/Order |
|:---|:---|
| 0-10% | +$96.06 |
| No Discount | +$68.11 |
| 10-20% | +$25.23 |
| 20%+ | **-$99.82** |

## 💡 Business Recommendations
1. Limit discounts to 10% maximum
2. Review Furniture category pricing
3. Increase inventory before November
